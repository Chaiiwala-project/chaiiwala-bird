# Chaiiwala-bird
Software team for [Chaiiwala's](https://www.chaiiwala.co.uk/) flappy-bird inspired videogame.

## Workspace Setup
To be able to work in this repository, you need to clone into it.
In your terminal, cd into a suitable repository, and use 
`git clone https://github.com/rezasroundtable/chaiiwala-bird`

After, you should cd into the newly clone repository.
`cd chaiiwala-bird` 

This will create a new directory containing the folders for this repository, and will move yourself into the directory.

You may also need to run
`git config --global user.name yourUsername` and
`git config --global user.email yourEmail`.
This allows your git to know who is trying to access the repository.

See CONTRIBUTING.md for advice on how to contribute to the repository.
